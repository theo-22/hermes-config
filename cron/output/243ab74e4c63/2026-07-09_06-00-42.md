Run the standing test-probe cleanup on Ted's Brain database. This is a preventive sweep to remove capability-test/probe thoughts that pollute semantic search.

## Procedure

### 1. Search for test/probe artifacts

Search Brain broadly for probe signatures using `mcp_open_brain_search_thoughts` with queries like:
- "capability check test content only"
- "capability probe"
- "Brain write tool test"
- "test thought from dashboard"
- "verifying capture works"
- "Thought captured successfully"

Also list recent thoughts (last 7 days) with `mcp_open_brain_list_thoughts(limit=30)` to catch anything not surfaced by search.

### 2. Identify confirmed probes

A thought is a confirmed probe when its content contains **at least two** of the following signals:

**Probe-action signals:**
- "capability check" / "capability probe"
- "Brain write tool test"
- "test thought"
- "verifying capture works"
- "Thought captured successfully"
- ":brain: Thought captured successfully." (bare success messages)

**Disclaimer signals:**
- "test content only" / "test only"
- "no operational finding"
- "harmless" / "harmless test"
- "no real finding"
- "probe only"

**Safe identification guardrails (DO NOT skip):**
- A real finding can legitimately mention "test" or "testing" — that alone is NOT a probe.
- Require MULTIPLE signals, not one.
- When a thought is ambiguous (mentions test but reads like a real finding/continuity note/documented behavior), **SKIP it and note it. Never delete.**
- Better to leave a probe than delete a real finding.

### 3. Delete confirmed probes

For each confirmed probe:
1. Note its id, preview, and source
2. Call `mcp_open_brain_delete_thought(thought_id=...)`
3. **Verify deletion** — re-search for the content or check the delete response. The API can return "deleted" even for nonexistent IDs (false-success bug). After each delete, confirm the thought is actually gone.

### 4. Log results

Write a timestamped log entry to `~/.hermes/profiles/lab-hermes/data/brain-sweep-logs/` in a file named `sweep_YYYY-MM-DD.md` with:
- Date/time of sweep
- How many thoughts examined
- How many matched as probes
- How many deleted (with IDs)
- How many skipped as ambiguous (with content previews and why)
- Any errors encountered

### 5. Report

If anything was deleted this run, also write an alert to `_AI_Inbox/` addressed "for Ted/CC" with a summary: how many probes found and removed, their source labels, and if any ambiguous cases were skipped.
