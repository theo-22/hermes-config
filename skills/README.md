# Skills Repo Notes

Run `./install-hooks` after cloning `~/Skills/` to reinstall the local pre-commit hook that checks `SKILL_INDEX.md` against the live skill directories.

`./skills-card-check` also validates required `SKILL.md` frontmatter for every shared skill: `name`, `description`, `category`, `write_mode`, `one_line_use`, and `fast_pick`. This keeps the shared skill index ready for future generation without making the index generated before the routing guidance can be reproduced faithfully.
